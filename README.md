# Diagnostico_Tese
Desenvolvido no PPGA-UFPB (Seminário de Tese I, Prof. Anielson), o instrumento transforma erros e recomendações de Freitas et al. (2019) em autoavaliação pré-qualificação. Criado pelas doutorandas Andressa, Dayanna, Jéssica e Mayne com apoio do Claude Sonnet 4.6 — código pela IA, conteúdo pelo grupo.
